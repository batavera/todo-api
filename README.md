# Todo App com API

Aplicação de lista de tarefas desenvolvida com HTML, CSS e JavaScript puro, consumindo dados de uma API pública com `fetch` e `async/await`.

## Funcionalidades

- Listagem de tarefas a partir de API externa
- Adição de novas tarefas
- Edição inline de tarefas
- Exclusão de tarefas
- Contador dinâmico de tarefas
- Validação de campo vazio
- Adição de tarefa com botão ou tecla Enter
- Tratamento de loading e erro durante requisições

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Fetch API
- JSONPlaceholder

## Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- consumo de API com `fetch`
- uso de `async/await`
- tratamento de erros
- manipulação de estado local
- renderização dinâmica no DOM
- operações de CRUD no frontend sem uso de frameworks
